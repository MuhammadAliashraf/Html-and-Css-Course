Lecture: How to Design Layout Using CSS Flexbox

1. What is Flexbox?

Flexbox is a one-dimensional layout model in CSS that makes it easier to design flexible and responsive layouts. It helps in aligning, spacing, and distributing elements inside a container.

2. Basic Syntax


.container {
  display: flex;
  flex-direction: row; /* or column */
  justify-content: center;
  align-items: center;
  
}
