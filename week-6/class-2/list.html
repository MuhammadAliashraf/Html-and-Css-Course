HTML Lecture: Lists

1. What are Lists in HTML?

Lists are used to group related items together in HTML. There are three main types of lists:

  • Ordered List - <ol>
  • Unordered List - <ul>
  • Description List - <dl>

2. Ordered List (<ol>)

Used when the order of items matters.


<ol>
  <li>Wake up</li>
  <li>Brush teeth</li>
  <li>Eat breakfast</li>
</ol>
  1. Wake up
  2. Brush teeth
  3. Eat breakfast

3. Unordered List (<ul>)

Used when the order doesn't matter. Default bullet points are shown. 


<ul>
  <li>Milk</li>
  <li>Bread</li>
  <li>Eggs</li>
</ul>
  • Milk
  • Bread
  • Eggs

4. Description List (<dl>)

Used for name-value pairs like definitions or FAQs.


<dl>
  <dt>HTML</dt>
  <dd>HyperText Markup Language</dd>

  <dt>CSS</dt>
  <dd>Cascading Style Sheets</dd>
</dl>
HTML
HyperText Markup Language
CSS
Cascading Style Sheets

5. Nesting Lists

You can nest lists inside other lists (e.g., sub-items).


<ul>
  <li>Fruits
    <ul>
      <li>Apple</li>
      <li>Banana</li>
    </ul>
  </li>
</ul>
  • Fruits
    • Apple
    • Banana

6. Styling Lists with CSS

You can remove or change bullets, numbers, and list style using list-style and other properties. 


ul {
  list-style-type: square;
  padding-left: 20px;
}

Common Values:

  • disc (default for ul)
  • circle
  • square
  • none
