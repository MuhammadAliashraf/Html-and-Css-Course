CSS Grid Lecture

Theory

CSS Grid Layout is a powerful two-dimensional layout system for the web. It allows you to create complex and responsive layouts with ease. Unlike Flexbox, which is primarily one-dimensional, Grid works in both rows and columns.

Key concepts include:

  • Grid Container: The parent element that establishes the grid.
  • Grid Items: The direct children of the grid container.
  • Grid Lines: The horizontal and vertical lines that divide the grid.
  • Grid Tracks: The spaces between grid lines (rows and columns).
  • Grid Areas: Named regions within the grid.

Practice Questions

  1. Create a grid with 3 columns and 2 rows.
  2. Use grid-template-areas to create a layout with a header, sidebar, content, and footer.
  3. Make the grid responsive, so it stacks items on smaller screens.
  4. Use grid-gap to add space between grid items.
  5. Use grid-column and grid-row to span items across multiple tracks.
